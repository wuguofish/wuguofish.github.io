這是什麼？

這是一份專為 PixAI 使用者撰寫的 SDXL 提示詞指南。

主要解決 AI 繪圖模型的「偏科問題」——模型很會畫可愛女生，但畫男生、風景、多人構圖等就需要特別技巧。這份指南整理了各種情境的實戰經驗和解法。

什麼是 PixAI？

PixAI 是一個 AI 繪圖平台，背後使用的算圖技術主要有三種：

  • DiT（Diffusion Transformer）— 最新的架構
  • SDXL（Stable Diffusion XL）— 本指南主要針對的技術
  • SD1.5（Stable Diffusion 1.5）— 較舊但仍有人使用

Stable Diffusion 是一個開源的文字到圖像生成模型，原始碼可以在 GitHub 和 Hugging Face 上取得，也可以在自己的電腦上跑。

PixAI 的價值：幫你省去自己購置硬體和架設軟體的成本，直接在網頁上就能使用各種模型生圖。
