Inpaint 與 Outpaint 局部編輯

📺 圖解版：本章節內容整理自 PixAI 使用心得分享簡報（第 37-38 頁）

🎬 官方教學影片：PixAI Inpaint/Outpaint Tutorial

概述

功能 用途 白話說明
Inpaint 局部重繪 「把這個區域擦掉，重新畫」
Outpaint 向外擴展 「把畫布變大，AI 幫你補上延伸的部分」

這兩個功能都是在既有圖片的基礎上，讓 AI「部分重新生成」。

Inpaint 局部重繪

什麼是 Inpaint？

Inpaint 讓你用「遮罩」（mask）標記想要修改的區域，AI 會只重畫那個部分，其他地方保持不變。

常見應用場景

場景 說明
🖐️ 修復手部崩壞 AI 畫的手常常有問題，用 Inpaint 局部重畫
👁️ 修復臉部問題 眼睛歪斜、表情怪異時局部修正
🚫 移除不想要的元素 背景多出的人、奇怪的物件
👗 更換服裝 只重畫服裝區域，換一套衣服
🎨 修改配色 只改頭髮顏色、眼睛顏色等
➕ 添加元素 在空白處加上飾品、道具

使用步驟

1. 上傳或選擇要修改的圖片
2. 用筆刷標記要重繪的區域（遮罩）
3. 輸入 prompt 描述你想要的結果
4. 調整 Denoising Strength
5. 點擊生成

Inpaint 的 Denoising Strength

和 HiRes 類似，Denoising 決定「改變程度」：

數值 效果 適用情境
0.3 ~ 0.5 輕微調整，保留大部分原樣 微調顏色、修復小瑕疵
0.5 ~ 0.7 中度改變，會重新詮釋 修復手部、換表情
0.7 ~ 1.0 大幅改變，幾乎重畫 完全替換元素

修手小技巧

手部崩壞是 AI 繪圖最常見的問題，Inpaint 修手流程：

1. 把崩壞的手部區域塗滿遮罩（畫大一點，包含手腕）
2. Prompt 加上手部描述，例如：
   - "beautiful detailed hands"
   - "correct hand anatomy"
   - "five fingers"
3. Denoising 設 0.6 ~ 0.8
4. 多生成幾次，挑最好的

Outpaint 向外擴展

什麼是 Outpaint？

Outpaint 讓你「擴大畫布」，AI 會根據現有圖片的內容，自動補上延伸的部分。

常見應用場景

場景 說明
📐 改變構圖比例 把正方形圖變成橫幅或直幅
🖼️ 擴展背景 讓風景延伸、增加環境空間
👤 顯示更多身體 半身像擴展成全身像
📱 適配不同尺寸 把圖片調整成適合手機/電腦桌布的比例

使用步驟

1. 上傳或選擇要擴展的圖片
2. 選擇擴展方向（上/下/左/右）
3. 設定擴展的大小
4. 輸入 prompt 描述延伸區域的內容（可選）
5. 點擊生成

Outpaint 注意事項

  • 風格一致性：AI 會嘗試維持原圖的風格，但不一定 100% 完美
  • 邊緣接合：有時候接合處會有明顯的痕跡，可以用 Inpaint 修補
  • 多次嘗試：Outpaint 的結果變化比較大，多生成幾次挑選

Inpaint vs Outpaint 比較

項目 Inpaint Outpaint
作用範圍 圖片內部 圖片外部
畫布大小 不變 變大
主要用途 修復、替換 擴展、延伸
需要遮罩 是 否（選擇方向即可）

實戰範例

範例 1：修復六指手

問題：AI 畫出了六根手指
解法：
1. Inpaint 遮罩塗滿整隻手
2. Prompt: "1girl, beautiful hand, five fingers, detailed fingers"
3. Negative: "extra fingers, mutated hands, bad anatomy"
4. Denoising: 0.7
5. 生成 4-8 張，挑最好的

範例 2：移除背景路人

問題：背景出現不想要的人物
解法：
1. Inpaint 遮罩塗滿那個人物
2. Prompt: 描述你想要的背景（如 "empty street, no people"）
3. Denoising: 0.8 ~ 1.0（需要完全替換）
4. 生成並檢查

範例 3：把半身像變全身像

問題：只有上半身，想看到全身
解法：
1. Outpaint 向下擴展
2. Prompt: "full body, standing, legs, feet, [服裝描述]"
3. 可能需要多次嘗試
4. 接合處不自然的話，用 Inpaint 修補

常見問題

Q：Inpaint 後遮罩區域和周圍格格不入？

A：

  • Denoising 調低一點，讓 AI 更參考周圍
  • 遮罩畫大一點，包含一些周圍區域
  • 在 prompt 中描述周圍的特徵

Q：Outpaint 的延伸部分風格不一致？

A：

  • 在 prompt 中強調原圖的風格特徵
  • 使用相同的模型和 LoRA 設定
  • 嘗試不同的 seed

Q：修手總是修不好？

A：

  • 遮罩要畫夠大，包含整個手掌和手腕
  • Prompt 要明確描述手的狀態（握拳、張開、比 V 等）
  • 試試加上 "realistic hands", "anatomically correct"
  • 有時候用 ControlNet 的 depth 或 openpose 可以幫助
