## 什麼是 LoRA？

**LoRA** （Low-Rank Adaptation）是一種「微調」技術，讓你可以在不重新訓練整個大模型的情況下，教會模型畫出特定的角色、風格或概念。

### 白話解釋

想像基礎模型是一個會畫畫的人，LoRA 就像是給他的「小抄」：

- **角色 LoRA**：「這個角色長這樣，髮色、眼睛、服裝要這樣畫」
- **風格 LoRA**：「用這種筆觸、配色來畫」
- **概念 LoRA**：「這個姿勢/場景/物件長這樣」

LoRA 檔案通常只有幾十到幾百 MB，比完整模型（幾 GB）小很多，方便下載和切換。

## LoRA 的權重概念

權重（Weight）決定 LoRA 對生成結果的影響程度：

| 權重值      | 效果                     |
|----------|------------------------|
| **0**    | LoRA 完全關閉，沒有任何影響       |
| **0.5**  | 效果減半，特徵較淡              |
| **1.0**  | 完整效果（預設值）              |
| **1.2+** | 加強效果，特徵更明顯             |
| **1.5+** | 過度強化，可能出現過擬合（臉崩、顏色過飽和） |

### 實際例子

假設你用一個「藍髮少女」的角色 LoRA：
```
權重 0.3 → 頭髮帶點藍色，但不太明顯
權重 0.7 → 藍髮特徵出現，但可能混合其他顏色
權重 1.0 → 標準的藍髮效果
權重 1.5 → 藍到發亮，可能連皮膚都染上藍色

```

**建議**：先從 1.0 開始測試，如果效果太強就往下調，太弱就往上加。

## LoRA 的種類

在 PixAI 和 CivitAI 上常見的 LoRA 類型：

| 類型          | 用途      | 例子                    |
|-------------|---------|-----------------------|
| **角色 LoRA** | 畫出特定角色  | 雷姆、2B、原神角色等           |
| **風格 LoRA** | 模仿特定畫風  | 厚塗風、水彩風、特定繪師風格        |
| **服裝 LoRA** | 特定服裝設計  | JK 制服、婚紗、盔甲等          |
| **姿勢 LoRA** | 特定動作或構圖 | 戰鬥姿勢、躺姿等              |
| **概念 LoRA** | 特定概念或效果 | 發光效果、特殊背景等            |
| **加速 LoRA** | 減少生成步數  | LCM、DMD2、PCM、Hyper-SD |

## LoRA 疊加技巧

你可以同時使用多個 LoRA！這是 PixAI 上很常見的玩法。

### 常見疊加組合

**組合 1：角色 + 風格**
```
角色 LoRA（權重 0.8）+ 風格 LoRA（權重 0.6）
→ 用特定畫風畫出特定角色

```

**組合 2：角色 + 服裝**
```
角色 LoRA（權重 1.0）+ 服裝 LoRA（權重 0.7）
→ 讓角色穿上不同的衣服

```

**組合 3：角色 + 風格 + 加速**
```
角色 LoRA（權重 0.8）+ 風格 LoRA（權重 0.5）+ LCM LoRA（權重 1.0）
→ 快速生成特定風格的角色圖

```

### 疊加注意事項

1. **總權重控制**：多個 LoRA 疊加時，建議每個都稍微降低權重，避免互相干擾
2. **數量限制**：一般建議 3-5 個以內，太多會讓結果混亂
3. **相容性**：同類型的 LoRA（如兩個角色 LoRA）疊加容易「打架」
4. **加速 LoRA 特殊**：加速 LoRA 一次只用一種，不要混用不同加速方案

### 權重調配範例

| LoRA 組合   | 建議權重                    |
|-----------|-------------------------|
| 單一角色 LoRA | 0.8 ~ 1.0               |
| 角色 + 風格   | 角色 0.7~0.9 / 風格 0.5~0.7 |
| 角色 + 服裝   | 角色 0.8~1.0 / 服裝 0.6~0.8 |
| 三個以上疊加    | 每個都降到 0.5~0.7           |

## 在 PixAI 上使用 LoRA

### 步驟

1. 在生成頁面找到「LoRA」區塊
2. 搜尋或瀏覽想要的 LoRA
3. 點擊加入，調整權重
4. 可重複加入多個 LoRA

### 小技巧

- **看 LoRA 說明**：很多 LoRA 作者會提供建議權重和觸發詞（trigger word）
- **觸發詞**：有些 LoRA 需要在 prompt 中加入特定詞彙才能啟動效果
- **基礎模型相容**：LoRA 是針對特定基礎模型訓練的。例如用 Illustrious XL 訓練的 LoRA，用在 Pony 家族模型上可能效果不如預期

## 常見問題

**Q：LoRA 效果不明顯怎麼辦？**

A：檢查以下幾點：

1. 權重是否太低？試著調高到 1.0 或以上
2. 有沒有加觸發詞？查看 LoRA 說明
3. 基礎模型對不對？用 Illustrious XL 訓練的 LoRA 用在 Pony 模型上效果可能會打折

**Q：LoRA 讓圖片崩壞了怎麼辦？**

A：通常是權重太高，試著： 1\. 降低權重到 0.6~0.8 2\. 減少同時使用的 LoRA 數量 3\. 確認 LoRA 和基礎模型相容

**Q：在哪裡找 LoRA？**

A：常見來源：

- PixAI 內建的 LoRA 庫
- [CivitAI](<https://civitai.com>)（最大的社群資源庫）
- [Hugging Face](<https://huggingface.co>)
