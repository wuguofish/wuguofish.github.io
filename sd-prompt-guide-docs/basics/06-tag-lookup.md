## 第一人稱視角（乙女 CG 風格）

用於畫出「從自己視角看他」的乙女遊戲 CG 風格圖片。

| 中文 | 英文 tag | 備註 |
|------|----------|------|
| 第一人稱女性視角 | `female pov` | ⚠️ 可能觸發 NSFW，記得加負面詞 |
| 「我」在畫面外 | `1girl out of frame` | 暗示女性存在但不入鏡 |
| 異性戀互動 | `hetero` | 強調男女互動 |
| 第一人稱的手 | `pov hands` | 畫面中出現「自己的手」 |
| 比一半愛心 | `half-heart hands` | 和畫面外的人合成愛心 |
| 摸對方的頭 | `hand on another's head` | |
| 摸對方的臉 | `hand on another's cheek` | |
| 牽手 | `holding hands` | |

**實測成功的組合**：
```
Positive: 1boy, male focus, female pov, 1girl out of frame, hetero, looking at viewer, [互動], [角色描述]
Negative: nsfw, nude
```
