# 在 PixAI 上訓練自己的 LoRA

> 學會使用 LoRA 之後，下一步就是訓練專屬於自己的 LoRA！PixAI 提供了免費的線上訓練功能，讓你不需要自己準備 GPU 也能輕鬆上手。

---

## 準備訓練圖片

訓練圖片的品質直接影響 LoRA 的效果，**重質不重量**是最重要的原則。

### 基本要求

- **數量**：建議 25～100 張（太少學不好，太多可能過擬合）
- **尺寸**：裁成正方形，邊長至少 768px
- **品質**：寧可高品質的圖片少一點，也不要一堆無關的雜圖

### 角色 LoRA 的特別注意事項

如果你要訓練的是角色 LoRA，還需要注意：

- **臉部清晰置中**：讓模型能好好學習角色的臉部特徵
- **避免遮擋**：盡量不要被手或其他東西遮擋臉部
  - 除非那是你想要訓練的特徵之一（例如角色招牌的手勢）
- **多角度**：最好每個角度都要至少 3 張
  - 正面、側面、45 度角、仰角、俯角

**常見錯誤**：只放正面照 → 模型只會畫正面，換角度就崩
**解決方法**：多準備不同角度的圖片，讓模型學會角色的立體結構

## 在 PixAI 上建立訓練

### 步驟 1：進入訓練頁面

在 PixAI 網站上找到「訓練」或「Train」功能（通常在側邊欄或個人頁面中）。

### 步驟 2：填寫基本資訊

| 欄位          | 說明                   | 建議                                             |
|-------------|----------------------|------------------------------------------------|
| **LoRA 名稱** | 你的 LoRA 顯示名稱         | 建議後綴訓練模型，方便辨識（例如：MyCharacter_illustriousXL）    |
| **模型類別**    | 選擇 SD 1.5、SDXL 或 DiT | 目前網站上最多的是 **SDXL**，**DiT**則是未來趨勢，建議至少訓練這兩個版本 |
| **模型主題**    | 選擇訓練用的基礎模型           | 選擇你預計要使用的模型，或至少是同家族的模型                         |
| **觸發詞**     | 啟動 LoRA 效果的關鍵字       | 輸入所有訓練圖片的**共通點**                               |

### 關於觸發詞

觸發詞是讓模型知道「現在要用這個 LoRA」的暗號。設定好的觸發詞應該是：

- 所有訓練圖片都有的共通特徵
- 不會跟其他常見詞彙混淆
- 好記、好打

**範例**：
```
角色名：hatsune miku, aqua hair, twin tails
風格：thick_lineart, painterly
服裝：frilly_maid_dress
```

### 步驟 3：上傳訓練圖片

把準備好的圖片上傳，PixAI 會自動處理後續的訓練流程。

## 訓練多個版本

PixAI 現在支援同一個 LoRA 訓練多個版本，這在以下情況很實用：

- 想要針對不同基礎模型各訓練一版（例如 Illustrious XL 和 NoobAI XL）
- 想要訓練適用不同技術模型的LoRA（例如 SDXL 和 DiT）
- 想要更新訓練資料後重新訓練

### 如何新增版本

1. 進入已完成訓練的模型頁面
2. 點擊「**新增版本**」按鈕
3. 或按右上角的選單，選擇「**重複使用資料集**」
4. 選擇新的基礎模型或調整參數
5. 開始訓練新版本

## 訓練後的測試

訓練完成後，記得要測試一下效果：

### 測試清單

- **基本測試**：用觸發詞生成，看角色/風格是否正確
- **角度測試**：測試不同視角（正面、側面、背面）
- **權重測試**：測試不同權重（0.5、0.8、1.0、1.2）的效果
- **組合測試**：跟其他 LoRA 疊加看會不會打架

### 如果效果不好

| 問題           | 可能原因       | 解決方法           |
|--------------|------------|----------------|
| 角色特徵不明顯      | 訓練圖片太少或太雜  | 精選更有代表性的圖片重新訓練 |
| 只能畫某個角度      | 訓練圖片角度太單一  | 補充不同角度的圖片      |
| 過擬合（所有圖都長一樣） | 訓練圖片太少或太相似 | 增加圖片多樣性        |
| 跟基礎模型風格不搭    | 訓練用的模型選錯   | 選擇正確的模型主題重新訓練  |
