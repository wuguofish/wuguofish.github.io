畫布寬(px) 畫布長(px)
計算尺寸 0 0
建議尺寸 0 0

* 計算尺寸是根據條件算出來的理論值。
但是實際上大多數手機的螢幕尺寸不會剛好是整數比，而會取一個接近的整數值作為真正的長寬。
在畫原型時，建議使用建議尺寸作為畫布尺寸。
或參考常用工具中的裝置尺寸查詢結果調整。
